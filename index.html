使用教程

手动打包步骤

  1. 下载Python脚本 po_to_mo_converter.py
  2. 确保已安装Python 3.6+
  3. 安装所需依赖： 
    pip install pyinstaller
  4. 打包为EXE： 
    pyinstaller --onefile --windowed --name POtoMOConverter po_to_mo_converter.py
  5. 生成的EXE文件在dist目录中

Android打包准备

  1. 安装Java JDK 11+
  2. 安装Android Studio和SDK
  3. 安装BeeWare工具： 
    pip install briefcase
  4. 初始化项目配置

使用打包脚本

Windows EXE打包：

  1. 下载build_exe.ps1脚本
  2. 右键脚本，选择"使用PowerShell运行"（需管理员权限）
  3. 脚本会自动安装所需依赖并打包
  4. 生成的EXE文件在dist_exe目录中

Android APK打包：

  1. 下载build_apk.ps1脚本
  2. 右键脚本，选择"使用PowerShell运行"（需管理员权限）
  3. 脚本会自动安装JDK、Android SDK和BeeWare工具
  4. 首次运行可能需要较长时间下载依赖
  5. 生成的APK文件在dist_apk目录中
注意： APK打包脚本需要稳定的网络连接，首次运行可能需要30分钟以上完成环境配置。
